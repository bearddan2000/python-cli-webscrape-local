# python-cli-webscrape-local

## Description
Scrapes a local nodejs service.

## Tech stack
- python

## Docker stack
- python:latest
- nodejs:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
